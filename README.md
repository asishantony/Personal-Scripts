# Personal Scripts

A curated collection of personal, production-safe automation scripts.

This repository contains utilities developed to streamline development
workflows, with a strong focus on safety, clarity, and repeatability.

## Principles

All scripts follow these principles:

- Dry run by default
- Explicit confirmation for destructive actions
- Clear terminal output
- Safe for shared repositories
- Easy to audit and modify

## Disclaimer

These scripts may perform destructive operations.
Always review output carefully before confirming any action.
